# Diferencia entre montar / desmontar un componente y re-renderizarlo\*\*

## **Montaje (Mount)**

- El componente se crea **por primera vez**
- React ejecuta toda la función del componente
- Se crea el **DOM desde cero**
- Se ejecutan los efectos: `useEffect(() => { ... }, [])` (mount)
- **Costo:** Alto (crear nodos DOM es costoso)

## **Re-renderizado (Re-render)**

- El componente **ya existe** en el árbol de React
- React ejecuta la función del componente de nuevo
- Compara el nuevo Virtual DOM con el anterior (**reconciliation**)
- Solo actualiza las **partes específicas del DOM real** que cambiaron
- **Costo:** Bajo a medio (dependiendo de cuánto cambió)

## **Desmontaje (Unmount)**

- El componente se **elimina completamente** del árbol
- Se ejecutan las funciones de limpieza: `useEffect(() => { return () => { ... } }, [])`
- Se elimina del DOM
- **Costo:** Medio
