1. Crear nuevo proyecto => movida
   Te asocia un id único: movida-ba47f
   (yo no he habilitado ni geminis IA ni analytics google para no complicarlo)

2. Nosotros solo vamos a compilacion => Authentication y firestore database

3. creo database en modo producción

4. setttings => agregar app => web
   Le meto tbe lo de hosting

5. npm install firebase

6. utils => firebase.js

7. npm install -g firebase-tools

   firebase login
   firebase init
   firebase deploy

8. Añado nuevo proveedor firebase authentication: correo electronico / contraseña

9. Compruebo acceso a dominio localhost

10. Pongo acceso a todo el mundo en modo escrituras en las reglas de firebase
    Cambio en las reglas de false a true para probar de momento, damos acceso público y le damos a publicar

11. Entender que la información deel usuario hay que almacenarla tanto en auth como en la database
