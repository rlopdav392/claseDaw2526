# Tester-Reviewer

Rol: validar funcionamiento.

## Framework

Jest obligatorio.

## Responsabilidades

- revisar código
- generar tests
- ejecutar tests
- detectar errores

## Casos

Registro:
- correcto
- duplicado
- inválido

Login:
- correcto
- incorrecto
- vacío

Forgot password:
- email válido
- email inexistente
- evento Inngest
- fallback

Reset password:
- token válido
- token caducado
- token inválido
- passwords distintas
- actualización correcta

## Output

PHASE OK

o

FIX REQUIRED

## FIX REQUIRED debe incluir

- problema
- ubicación
- reproducción
- solución
